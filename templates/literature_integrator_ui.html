🏗️ 系統架構概覽

模組 檔案 功能說明
📡 Crawler arxiv_crawler.py arXiv REST API 爬蟲，支援關鍵字 + 日期範圍查詢
📡 Crawler biorxiv_crawler.py bioRxiv / medRxiv (Europe PMC) 爬蟲，支援 2020–2026 歷史資料
🔗 Resolver paper_resolver.py 貼上任意 URL / DOI 自動解析論文元數據
🗄️ Database db_manager.py SQLite + SQLAlchemy ORM，論文資料持久化
🤖 LLM llm_analyzer.py 多後端 LLM 分析：Qwen2.5 / Gemini 2.0 Flash
📓 Integrator notion_client.py Notion API 整合，每日自動建立日誌子頁面
📧 Notifier notifier.py Gmail SMTP Email + LINE Messaging API 每日推播
⏰ Scheduler daily_job.py 每日自動化排程，支援 daily / historical 兩種模式
📊 Dashboard app.py Streamlit 互動儀表板，深色玻璃質感 UI
