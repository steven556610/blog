> Accessing the Mainframe API

我建立了一個 API，你可以試著用 Python 的 requests 來抓取資料。

1. API Endpoint

GET /api/v1/get-code

2. Python 範例程式碼

import requests

response = requests.get('http://127.0.0.1:5000/api/v1/get-code')
data = response.json()

print(f"Author: {data['author']}")
print(f"Code: \n{data['snippet']}")
